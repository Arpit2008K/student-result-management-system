# Student Result Management System (Java)

A Java console-based application that calculates a student’s total marks, percentage, and grade based on subject scores.

## Features
- Accepts number of subjects
- Takes marks for each subject (out of 100)
- Calculates total marks and percentage
- Assigns grade based on percentage

## Grade Criteria
- A : Percentage ≥ 75
- B : Percentage ≥ 60
- C : Percentage ≥ 40
- F : Percentage < 40

## Concepts Used
- Scanner class
- Arrays
- Loops
- Conditional statements

## How to Run
1. Compile the program using `javac Result.java`
2. Run using `java Result`

## Purpose
This project was created as a first GitHub repository while learning Java fundamentals.
